How Binary Search Trees Work

A Binary Search Tree (BST) is a special type of binary tree that keeps elements in a sorted order. Each node contains a value, and it can have up to two children: a left child and a right child.

  • Left Child: If a new value is less than the current node's value, it goes to the left.
  • Right Child: If a new value is greater than the current node's value, it goes to the right.

This rule applies recursively at each node, which helps maintain an ordered structure and allows for efficient operations.

  • Insert: Traverse from the root, moving left or right depending on comparisons, and place the new value in the correct position.
  • Find: Navigate the tree using the same logic to quickly locate a value.
  • Delete: Remove a node and adjust the tree to keep it properly ordered.
